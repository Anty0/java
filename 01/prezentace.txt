# Java je

* jazyk třetí generace (stejně jako C, C#, Pascal ...)
* objektově orientovaný jazyk - OOP (stejně jako C++, C#, Delphi ...)
* jednodušší než C++, srovnatelná s C#
* Case sensitive - rozlišuje velikost písmen

-----

# Rychlost

* pomalejší než kompilované jazyky (C, C++)
* srovnatelná s ostatními Just in Time kompilovanými jazyky (C#)
* rychlejší než jazyky bez kompilátoru (Python, Ruby, PHP)

-----

# JVM

* umožňuje multiplatformní programy
* výstupem překladače je mezikód (bytecode), který je JVM zpracováván
* provádí překlad mezikódu do strojového kódu 
* je součástí JRE

-----

# JRE vs JDK

* JRE < JDK
* JRE umožňuje Javu spustit
* JDK umožňuje Javu vyvíjet

# Hello world II.

* vytvoření vlastní složky:  mkdir "název"
* otevření složky: cd "název"
* vypsání obsahu složky: ls
* vytvoření souboru: touch hello.java
* editace souboru: nano hello.java
* uložení změn: CTRL + O
* zavření editoru: CTRL + X
* přeložení programu: javac hello.java
* spuštění programu" java Hello
* odhlášení: logout
